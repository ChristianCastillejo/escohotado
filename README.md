# README

## Intro

Open source official website for the Spanish philosopher Antonio Escohotado

**Frontend:**

- React
- React Router
- Redux
- Axios

**Backend:**

- Ruby on Rails
- PostgreSQL

**Auth:**

- JWT

## Usage

**Run the server:**

`$ cd escohotado-server`

`$ rails db:creater`

`$ rails db:migrate`

`$ rails s -p 4000`

**Run the client:**

`$ cd escohotado-client`

`$ yarn install`

`$ yarn start`

_You need to run server and client at the same time._
